Data analysis in R

Friday March 3, 10.15am

Location: City Terrace 10

You’ll learn how to use the dplyr package to sort, filter, join and carry out some other basic functions in R to identify trends in your data for storytelling. We’ll also make some simple charts with ggplot2.
